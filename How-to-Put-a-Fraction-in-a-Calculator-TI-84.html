How to Put a Fraction in a Calculator TI-84: Step-by-Step Guide

Entering Fractions on a TI-84 Plus

1. Open the Fraction Menu

To input a fraction, follow these steps:

  1. Press the ALPHA key (located in the upper-left corner of the calculator).
  2. Press the Y= key (just below the calculator screen). This opens the FRAC menu, which contains fraction-related functions.

2. Select the Fraction Format

  • The first option, n/d, represents a standard fraction template.
  • Press ENTER to select it.

3. Input the Numerator and Denominator

  1. Type the numerator (the number on top of the fraction).
  2. Press the down arrow (▼) key to move to the denominator field.
  3. Enter the denominator (the number on the bottom of the fraction).
  4. Press ENTER to confirm.

You can now perform calculations using this fraction as part of your equation.

Entering Mixed Numbers on a TI-84 Plus

If you need to enter a mixed number (e.g., 2 ¾), use the following method:

  1. Press ALPHA → Y= to open the FRAC menu.
  2. Select the second option, Un/d (which stands for mixed number format).
  3. Enter the whole number first, then navigate using arrow keys to input the numerator and denominator.
  4. Press ENTER to finalize.

Converting Fractions to Decimals (and Vice Versa)

The TI-84 Plus can convert fractions into decimals and improper fractions into mixed numbers.

  • To switch between fractions and decimals, press MATH, then choose 1:Frac or 2:Dec.
  • To convert an improper fraction to a mixed number, use the FRAC menu and select the third option.

Setting the Calculator to Display Fractions by Default

By default, the TI-84 Plus displays results in decimal form. You can change this setting to show fractions instead:

  1. Press the MODE button.
  2. Scroll down to the Fraction Display options.
  3. Choose one of the following:
    • n/d: Displays answers as improper fractions.
    • Un/d: Shows results as mixed numbers.
    • Auto: Matches the format of your input.
  4. Press ENTER to confirm your selection.
